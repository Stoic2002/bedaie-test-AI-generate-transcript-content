# Key Features

The AI Content Studio is designed to be a comprehensive tool for creators to brainstorm and draft video content. Below are its primary features.

## 1. Multi-Provider AI Engine
Toggle between the world's most powerful AI models without leaving the app:
- **Google Gemini**: Optimized for fast, creative content generation.
- **Anthropic Claude**: Excellent for structured scripts and complex topics.
- **OpenAI GPT-4**: Industry-standard logic and quality.
- **OpenRouter Support**: Access hundreds of additional models via OpenRouter.

## 2. Advanced Content Generation
The generation engine doesn't just write a script; it builds a full production scene breakdown:
- **Automatic Scene Creation**: Splits content into logical visual segments.
- **Visual Descriptions**: Provides cues for editors and videographers.
- **Voiceover Scripts**: Ready-to-read narration for each scene.
- **On-Screen Text cues**: Highlights key text that should appear on screen.
- **Technical Cues**: Suggestions for transitions and background music (BGM).

## 3. Localization and i18n
The studio supports a multi-language workflow:
- **UI Localization**: Switch the interface between English, Indonesian, and Malaysian.
- **Output Localization**: Generate content in a language different from your UI choice (e.g., Use English UI but generate Indonesian scripts).

## 4. Smart Templates
Accelerate your workflow with pre-built content templates:
- **Marketing Videos**: Focused on conversions and product launches.
- **Educational Clips**: Optimized for information retention and clarity.
- **Social Media Reels**: High-energy, vertical-friendly short-form content.
- **Product Demos**: Balanced feature highlights and benefits.
- **Explainer Videos**: Clear, friendly walkthroughs for new concepts.

## 5. Variations & Exporting
- **Generate Variations**: Create up to 3 distinct versions of a script in a single click to find the perfect angle.
- **Copy & Download**: Quickly copy the entire scene breakdown to your clipboard or download it as a `.txt` file for use in production.

## 6. History Tracking
All generated content is saved to your account, allowing you to revisit, copy, or download past scripts from the **History** tab.
